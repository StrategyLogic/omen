# Situation Sources

This directory stores source documents for scenario modeling.

Pipeline role:

- source: files in this directory
- data: `omen analyze situation` outputs structured artifacts to `data/scenarios/`
- output: `omen simulate --scenario ...` writes simulation artifacts to `output/`

Suggested fields for each situation document:

- company current state
- appointment context
- key dilemma and constraints
- resistance signals
- desired strategic outcomes

Example file name:

- `nokia-elop-2010.md`

# Strategic Actor Concepts

This guide defines the Spec 7 OSS baseline concept model.

## Core Term Split

- CLI and artifact contract term: `actor`
- UI and documentation product term: `Strategic Actor`

## Scope for Spec 7

Spec 7 delivers an input-layer capability:

- local actor ontology generation
- persona and status artifacts
- graph and timeline UI rendering from local artifacts
- deterministic validation reports

Out of scope for Spec 7 input-layer delivery:

- scenario-result causal explanation as a release artifact
- counterfactual comparison workflows as a release artifact
- founder plugin runtime semantics

## Founder as Source Domain

Founder case markdown is accepted as actor-source input material in Spec 7.
The public contract remains actor-named (`actor_ontology.json`, `output/actors`).
